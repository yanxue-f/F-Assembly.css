    • 基本单选框

    最基本的用法。

                        
<div class="radio">
    <input type="radio" value="" id="radio"  hidden >
    <label for="radio" class="radio-circular">Radio</label>
</div>

    单选框组合

    用相同的name将radio组合起来。采用相同的name互斥

                        
<div class="radio">
    <input type="radio" value="" id="compose-a" name="compose" hidden >
    <label for="compose-a" class="radio-circular">A</label>
</div>
<div class="radio">
    <input type="radio" value="" id="compose-b" name="compose" hidden >
    <label for="compose-b" class="radio-circular">B</label>
</div>
<div class="radio">
    <input type="radio" value="" id="compose-c" name="compose" hidden >
    <label for="compose-c" class="radio-circular">C</label>
</div>
<div class="radio">
    <input type="radio" value="" id="compose-d" name="compose" hidden >
    <label for="compose-d" class="radio-circular">D</label>
</div>
<style>
    .radio:not(:last-child){
        margin-right: 10px;
    }
</style>

    自定义单选框

    通过改变radio或radio-block下的变量：--radio ，--radio-color ，--radio-shadow的值来改变单选框的样式。

                        
<div class="radio">
    <input type="radio" value="" id="Radio-default"  hidden >
    <label for="Radio-default" class="radio-circular">Radio-default</label>
</div>
<div class="radio">
    <input type="radio" value="" id="Radio-size"  hidden >
    <label for="Radio-size" class="radio-circular">Radio-size</label>
 </div>
<div class="radio">
    <input type="radio" value="" id="Radio-color"  hidden >
    <label for="Radio-color" class="radio-circular">Radio-color</label>
</div>
<style>
    .radio:nth-child(2){
        --radio:30px;
}
    .radio:nth-child(3){
        --radio-color:rgb(255, 24, 24);
        --radio-shadow:rgba(255, 24, 24,0.2) ;
}
</style>

    禁用状态

    radio不可用。

                        
<div class="radio">
    <input type="radio" value="" id="disabled-1" hidden disabled>
    <label for="disabled-1" class="radio-circular">Disabled</label>
</div>
<div class="radio">
    <input type="radio" value="" id="disabled-2" hidden disabled checked>
    <label for="disabled-2" class="radio-circular">Disabled+checked</label>
    </div>
<style>
    .radio:not(:last-child){
        margin-right: 10px;
    }
</style>

    垂直单选框

    使用radio-block使单选框垂直排列。

    <input type="radio" value="" id="Option-1" name="Option" hidden >

</div>
<div class="radio-block">
    <input type="radio" value="" id="Option-2" name="Option" hidden >

</div>
