    • 基本多选框

    最基本的用法。

                        
<label  class="checkbox">
    <input type="checkbox" value=""    >
    <div class="checkbox-frame"></div>
    checkbox
</label>

    多选框组合

    用相同的name将checkbox组合起来。采用相同的name互斥

                        
<label  class="checkbox">
    <input type="checkbox" value=""  name="compose">
    <div class="checkbox-frame"></div>
    A
</label>
<label class="checkbox">
    <input type="checkbox" value=""  name="compose">
    <div class="checkbox-frame"></div>
    B
</label> 
<label class="checkbox">
    <input type="checkbox" value=""  name="compose">
    <div class="checkbox-frame"></div>
    C
</label>
<label class="checkbox">
    <input type="checkbox" value=""  name="compose">
    <div class="checkbox-frame"></div>
    D
</label>
<style>
    .checkbox:not(:last-child){
        margin-right: 10px;
    }
</style>




    多选框大小

</label>

    自定义多选框

    通过改变checkbox或checkbox-block下的变量：--checkbox ，--checkbox-color ，--checkbox-shadow的值来改变多选框的样式。通过改变checkbox或checkbox-block下gap属性改变文字距方点距离。

                        
<label  class="checkbox">
    <input type="checkbox" value="">
    <div class="checkbox-frame"></div>
    checkbox-default
</label>
<label  class="checkbox">
    <input type="checkbox" value="">
    <div class="checkbox-frame"></div>
    checkbox-size
</label>
<label  class="checkbox">
    <input type="checkbox" value="">
    <div class="checkbox-frame"></div>
    checkbox-color
</label>
<style>
    .checkbox:nth-child(2){
        --checkbox:30px;
}
    .checkbox:nth-child(3){
        --checkbox-color:rgb(255, 24, 24);
        --checkbox-shadow:rgba(255, 24, 24,0.2) ;
}
</style>

    禁用状态

    checkbox不可用。

                        
<label  class="checkbox">
    <input type="checkbox" value="" disabled>
    <div class="checkbox-frame"></div>
    <span>Disabled</span>
</label>
<label  class="checkbox">
    <input type="checkbox" value="" disabled checked>
    <div class="checkbox-frame"></div>
    <span>Disabled+Checked</span>
</label>
<style>
    .checkbox:not(:last-child){
        margin-right: 10px;
    }
</style>

    垂直多选框

    使用checkbox-block使多选框垂直排列。

                        
<label class="checkbox-block">
    <input type="checkbox" value="" name="Option">
    <div class="checkbox-frame"></div>
    Option-1
</label>
<label class="checkbox-block">
    <input type="checkbox" value="" name="Option">
    <div class="checkbox-frame"></div>
    Option-2
</label>
<label class="checkbox-block">
    <input type="checkbox" value="" name="Option">
    <div class="checkbox-frame"></div>
    Option-3
</label>
<label class="checkbox-block">
    <input type="checkbox" value="" name="Option">
    <div class="checkbox-frame"></div>
    Option-4
</label>
<style>
    .checkbox-block:not(:last-child){
        margin-right: 10px;
    }
</style>
