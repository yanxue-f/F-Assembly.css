    • 基本多选框

    最基本的用法。

                        
<div class="checkbox">
    <input type="checkbox" value="" id="checkbox"  hidden >
    <label for="checkbox" class="checkbox-frame">checkbox</label>
</div>

    多选框组合

    用相同的name将checkbox组合起来。采用相同的name。

                        
<div class="checkbox">
    <input type="checkbox" value="" id="compose-a" name="compose" hidden >
    <label for="compose-a" class="checkbox-frame">A</label>
</div>
<div class="checkbox">
    <input type="checkbox" value="" id="compose-b" name="compose" hidden >
    <label for="compose-b" class="checkbox-frame">B</label>
</div>
<div class="checkbox">
    <input type="checkbox" value="" id="compose-c" name="compose" hidden >
    <label for="compose-c" class="checkbox-frame">C</label>
</div>
<div class="checkbox">
    <input type="checkbox" value="" id="compose-d" name="compose" hidden >
    <label for="compose-d" class="checkbox-frame">D</label>
</div>
<style>
    .checkbox:not(:last-child){
        margin-right: 10px;
    }
</style>

    自定义多选框

    通过改变checkbox或checkbox-block下的变量：--checkbox ，--checkbox-color ，--checkbox-shadow的值来改变多选框的样式。

                        
<div class="checkbox">
    <input type="checkbox" value="" id="checkbox-default"  hidden >
    <label for="checkbox-default" class="checkbox-frame">checkbox-default</label>
</div>
<div class="checkbox">
    <input type="checkbox" value="" id="checkbox-size"  hidden >
    <label for="checkbox-size" class="checkbox-frame">checkbox-size</label>
 </div>
<div class="checkbox">
    <input type="checkbox" value="" id="checkbox-color"  hidden >
    <label for="checkbox-color" class="checkbox-frame">checkbox-color</label>
</div>
<style>
    .checkbox:nth-child(2){
        --checkbox:30px;
}
    .checkbox:nth-child(3){
        --checkbox-color:rgb(255, 24, 24);
        --checkbox-shadow:rgba(255, 24, 24,0.2) ;
}
</style>

    禁用状态

    checkbox不可用。

                        
<div class="checkbox">
    <input type="checkbox" value="" id="disabled-1" hidden disabled>
    <label for="disabled-1" class="checkbox-frame">Disabled</label>
</div>
<div class="checkbox">
    <input type="checkbox" value="" id="disabled-2" hidden disabled checked>
    <label for="disabled-2" class="checkbox-frame">Disabled+checked</label>
    </div>
<style>
    .checkbox:not(:last-child){
        margin-right: 10px;
    }
</style>

    垂直多选框

    使用checkbox-block使多选框垂直排列。

    <input type="checkbox" value="" id="Option-1" name="Option" hidden >

</div>
<div class="checkbox-block">
    <input type="checkbox" value="" id="Option-2" name="Option" hidden >

</div>

    <input type="checkbox" value="" id="Option-3" name="Option" hidden >

<div class="checkbox-block">
    <input type="checkbox" value="" id="Option-4" name="Option" hidden >
