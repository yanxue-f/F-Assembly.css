Button 按钮

常用的各种类型的按钮

基础按钮

在button标签中添加button类来实现基础按钮，而后通过添加各种不同的类名来实现不同的按钮样式。

plain类代表朴素类型的按钮，round类代表圆角类型按钮，circle代表圆形图标按钮（不包含文字需要插入相应的图片）。 button-default类代表默认按钮(透明)，button-main类代表圆角主要按钮（蓝色），button-success代表圆形成功按钮（绿色），button-info代表圆形信息按钮（灰色） button-warn类代表警告按钮(橙色)，button-danger类代表危险按钮(红色)。 

                
<div class="tr">
    <button class="button button-default">Default</button>
    <button class="button button-main">Primary</button>
    <button class="button button-success">Success</button>
    <button class="button button-info">Info</button>
    <button class="button button-warn">Warning</button>
    <button class="button button-danger">Danger</button>
    <button class="button button-default">中文</button>
</div>
<!-- 朴素按钮 -->
<div class="tr">
    <button class="button button-default plain">Plain</button>
    <button class="button button-main plain">Primary</button>
    <button class="button button-success plain">Success</button>
    <button class="button button-info plain">Info</button>
    <button class="button button-warn plain">Warning</button>
    <button class="button button-danger plain">Danger</button>
</div>
<!-- 圆角按钮 -->
<div class="tr">
    <button class="button button-default round">Round</button>
    <button class="button button-main round">Primary</button>
    <button class="button button-success round">Success</button>
    <button class="button button-info round">Info</button>
    <button class="button button-warn round">Warning</button>
    <button class="button button-danger round">Danger</button>
</div>
<!-- 圆形图标按钮 -->

禁用状态

你可以使用 disabled 属性来定义按钮是否被禁用。

通过为button类元素添加disabled属性实现禁用按钮

                
<!-- 禁用 -->
<div class="tr">
    <button class="button button-default" disabled>Default</button>
    <button class="button button-main" disabled>Primary</button>

文本按钮

没有边框和背景色的按钮。

通过为button类元素添加button-text类实现纯文本按钮

                
<!-- 文本按钮 -->
<div class="tr">
    <button class="button button-text">Text button</button>
    <button class="button button-text" disabled>Text button</button>
</div>

图标按钮

使用图标为按钮添加更多的含义。 你也可以单独使用图标不添加文字来节省显示区域占用。

通过为button类元素button-icon，并插入相应的图片来为按钮添加图标。

                
<!-- 图标按钮 -->
<div class="tr">
    <button class="button button-icon"><img src="./icon/bianji-w.png" alt="edit"></button>
    <button class="button button-icon"><img src="./icon/fenxiangxiao-w.png" alt="share"></button>
    <button class="button button-icon"><img src="./icon/shanchu-w.png" alt="delete"></button>
    <button class="button button-icon"><img src="./icon/sousuo-w.png" alt="search"><span>Search</span></button>
    <button class="button button-icon"><span>Upload</span><img src="./icon/shangchuan-w.png" alt="upload"></button>
    <button class="button button-icon" disabled><span>Upload</span><img src="./icon/shangchuan-w.png" alt="upload"></button>
</div>

加载状态按钮

点击按钮处于加载状态，向用户反馈数据的情况。

通过为button类元素button-loading，并插入相应的图像来实现加载状态按钮。

                
<!-- 加载中 -->
<div class="tr">
    <button class="button button-loading"><img src="./icon/jiazai-w.png" alt="loading"><span>Loading</span></button>
</div>

各种尺寸的尺寸按钮

除了默认的大小，按钮组件还提供了几种额外的尺寸可供选择，以便适配不同的场景。

通过为button类元素添加相应的类实现不同大小的图标，除默认外还包含三种大小：button-medium，button-small，button-mini。

                
<!-- 不同大小的按钮 -->
<div class="tr">
    <button class="button button-default">Default</button>
    <button class="button button-default button-medium">Medium</button>
    <button class="button button-default button-small">Small</button>
    <button class="button button-default button-mini">Mini</button>
</div>
